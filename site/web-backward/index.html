数字記憶テスト（逆唱）

研究者が指定した日本語または英語で読み上げられる数字を、聞こえた順番とは逆に入力する課題です。

数字の読み上げ音声は研究者が指定した言語に固定されています。

実施方法

  • 数字を1桁ずつ読み上げます。画面が入力可能になるまでキーを押さないでください。
  • 最後のビープ音のあと、聞こえた数字を逆の順番で入力します。
  • 例：音声が「2・7・4」の場合は「472」と入力します。
  • 回答時間は桁数に応じて8～20秒です。時間になると入力済みの内容が自動送信されます。
  • メモ、外部機器、ブラウザの戻る・再読み込みは使用しないでください。
  • 練習3試行のあと、本試行21試行（3～9桁×各3試行）を行います。

氏名・学籍番号・メールアドレスは入力しないでください。回答と参加者IDはサーバーへ送信されず、この端末にCSVとして保存されます。

開始前に音声を確認してください。

練習

3桁の練習を3試行行います。聞こえた数字を逆順で入力してください。

練習 0 / 3

準備ができたら開始してください。

本試行

3～9桁を各3試行、合計21試行行います。本試行中は正誤を表示しません。

本試行 0 / 21
