参加者リンクの作成

参加者画面の開始言語と読み上げ音声を個別に選び、確定済みのプロトコル・課題・タイミングを固定したリンクを作成します。

このページは認証された管理画面ではありません

GitHub Pages上では誰でも閲覧できます。秘密、パスワード、参加者ID、回答、個別の研究情報を入力・掲載しないでください。リンクは署名付きでも一回限りでもありません。

固定参加者リンク

音声条件は参加者側で変更できず、順唱から逆唱にも引き継がれます。

参加者画面を別タブで開く

匿名IDはリンクへ追加せず、課題画面で本人に入力してもらいます。

変更できない研究仕様

音声言語以外の試行数、刺激列、提示間隔、回答期限、採点、順唱→逆唱の順序は、このページから変更できません。

実施時の責任範囲

  • 研究説明・同意・撤回・連絡先は、承認済みの研究手続きで別途管理します。
  • 順唱と逆唱のCSVをそれぞれ保存し、承認済みの制限領域へ移動します。Pagesは結果を受信しません。
  • 公開静的サイトでは刺激列と音声を隠せないため、事前曝露は運用上管理します。
  • 2026-07-20に、固定音声 ja-digits-aligned-wav-v3、en-us-digits-aligned-wav-v1、beep.wav の公開が承認されました。公開時はタグ内容を照合し、手動確認ゲートを実行します。このプロジェクト承認は第三者の法的保証ではありません。
